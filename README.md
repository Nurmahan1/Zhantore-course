# 🐍 zhon1 — Python Бағдарламалау Курсы

Қазақ тіліндегі заманауи Python бағдарламалау курсы.  
Бұл жоба оқушылар мен бастаушыларға Python тілін нөлден бастап үйренуге арналған интерактивті веб-платформа болып табылады.

## 📌 Жоба туралы

`zhon1` — HTML, CSS және JavaScript көмегімен жасалған онлайн оқу платформасы.  
Сайт ішінде:

- Python сабақтары
- Интерактивті компилятор
- AI көмекші
- Прогресс бақылау
- Сертификат жүйесі
- Көптілді интерфейс
- Пайдаланушы профилі

сияқты мүмкіндіктер бар.
